Tools & IDEs

The right tools make you 10x more productive. Master Git, VS Code, Postman, Figma, and the full developer toolkit.

💙
VS Code
Editor
Free, extensible editor from Microsoft with IntelliSense, debugger, and terminal built in.
🔧
Git
Version Control
Track changes, work in branches, collaborate with others on the same codebase.
🐙
GitHub
Hosting
Host repos, PR reviews, Actions CI/CD pipelines, Issues, and code collaboration.
📮
Postman
API Testing
Test REST APIs, manage environments, write automated test suites, generate docs.
🎨
Figma
Design
Collaborative UI design, prototyping, design systems and handoff to developers.
🔍
Chrome DevTools
Browser
Debug JS, inspect CSS, analyse network requests, and profile rendering performance.
⚫
Warp Terminal
Terminal
AI-powered terminal with splits, command autocomplete and modern UX.
🟢
WebStorm
Editor
JetBrains IDE with deep JS/TS support, inline debugger, and smart refactoring.
⚡
Insomnia
API Testing
REST and GraphQL client with env variables, collections, and code snippet generation.
🐚
Zsh + Oh My Zsh
Terminal
Shell with themes, plugins, git integration, and tab auto-completion.
🖼️
Storybook
UI Dev
Develop, document, and test UI components in isolation. Build design systems.
🔀
GitLab
Hosting
Self-hostable alternative to GitHub with integrated CI/CD and DevOps tooling.

Essential Git Commands

Commands every developer must know by heart.

git init
Initialise new repository
git clone [url]
Clone a remote repository
git add . && git commit -m "msg"
Stage all and commit
git checkout -b feature/name
Create and switch branch
git pull origin main
Pull latest changes
git rebase -i HEAD~3
Interactive rebase
git stash pop
Apply stashed changes
git log --oneline --graph
Visual commit history
